## 🧠 About

AI engineer focused on **search, retrieval, and LLM infrastructure**. I build systems that go beyond the demo — tuned for production traffic, real query distributions, and scale.

- ⚡ **Inference APIs** — LLM serving at scale with vLLM, continuous batching, KV cache tuning, and async request handling
- 🔍 **Retrieval Pipelines** — hybrid BM25 + vector search with cross-encoder reranking and end-to-end relevance tuning
- 🔁 **Event-Driven Systems** — high-throughput Kafka/RabbitMQ pipelines with scalable consumers and fault-tolerant async processing
- 🧠 **Semantic Search** — embedding models, vector indexing, and relevance scoring tuned for real query distributions
